# Excel Sales Dashboard Project

This project is based on retail sales data analysis using Microsoft Excel. The objective of this project is to analyze sales, profit, customer segments, and regional performance using pivot tables, charts, and dashboard visualizations.

---

# Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Visualization

---

# Dashboard Analysis

The dashboard includes:

- Sales by Region
- Profit by Category
- Sales by Segment
- Interactive Slicers
- Sales Trend Visualizations

---

# Key Insights

- West region generated highest sales.
- Technology category showed better profits.
- Consumer segment contributed highest sales.
- Interactive slicers help filter dashboard data dynamically.
